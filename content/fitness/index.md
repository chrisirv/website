This page pulls data directly from the <strong>Strava API</strong>.  Every few hours, a <strong>GitHub Action</strong> wakes up, grabs any updates, and saves them to a data file. <strong>Hugo</strong> then does any converting and using <strong>Leaflet.js</strong> plots my last starting point on a map.
